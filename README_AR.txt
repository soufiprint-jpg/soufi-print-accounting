Soufi Print Accounting - بدون تسجيل دخول

هذه النسخة هي نفس النسخة التي فيها:
- لوحة الإعدادات
- الحقول المخصصة
- الشعار
- كل الأقسام

لكن تم إلغاء:
- شاشة تسجيل الدخول
- اسم المستخدم
- كلمة السر

التشغيل:
1) فك الضغط
2) افتح index.html من مجلد ثابت


التعديل الجديد:
- شريط بحث في الأصناف
- شريط بحث في المدخول/المبيعات
- ترتيب الأصناف أبجديًا عربي/إنجليزي
- إمكانية إضافة صورة بجانب كل اسم منتج

- تم تحسين إضافة صورة المنتج بحيث تظهر معاينة مباشرة وتُحفظ مع المنتج داخل المتصفح.

- تم إصلاح خانة رفع الصورة: زر اختيار صورة ظاهر الآن مع اسم الملف ومعاينة مباشرة.
